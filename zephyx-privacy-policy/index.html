Politique de Confidentialité - Zephyx

Dernière mise à jour : 25/01/2025

Introduction

Cette politique de confidentialité décrit comment l'extension Chrome Zephyx gère vos données. Nous nous engageons à protéger votre vie privée et à être transparents sur nos pratiques de traitement des données.

Collecte et Utilisation des Données

Données Collectées

L'extension Zephyx collecte uniquement :

  • Les fichiers Excel (.xlsx, .xls) que vous importez volontairement
  • Vos préférences d'utilisation (colonnes sélectionnées, filtres, thème choisi)

Stockage des Données

  • Toutes les données sont stockées exclusivement en local dans votre navigateur Chrome via le système de stockage intégré (Chrome Storage API).
  • Aucune donnée n'est transmise ou stockée sur des serveurs externes.
  • Les données restent sur votre appareil et sont accessibles uniquement par vous.

Utilisation des Données

Les données sont utilisées uniquement pour :

  • Afficher et gérer vos fichiers Excel dans l'interface de l'extension
  • Sauvegarder vos préférences d'affichage
  • Permettre les fonctionnalités de filtrage et de tri

Partage des Données

Absence de Partage

  • Nous ne vendons pas vos données.
  • Nous ne partageons pas vos données avec des tiers.
  • Nous ne transférons pas vos données vers des serveurs externes.
  • Nous n'utilisons pas vos données à des fins publicitaires ou de marketing.

Sécurité des Données

Mesures de Protection

  • Les données sont stockées localement avec le niveau de sécurité fourni par Chrome.
  • L'extension fonctionne en mode "sandbox" isolé.
  • Aucune connexion externe n'est établie pour le traitement des données.

Vos Droits

En tant qu'utilisateur, vous avez le contrôle total sur vos données :

  • Vous pouvez supprimer toutes les données stockées à tout moment via l'interface de l'extension.
  • Vous pouvez exporter vos données au format PDF.
  • Vous pouvez désinstaller l'extension, ce qui supprimera toutes les données stockées.

Bibliothèques Tierces

L'extension utilise les bibliothèques suivantes, incluses dans le package :

  • xlsx.min.js : pour le traitement des fichiers Excel
  • html2pdf.bundle.min.js : pour la génération de PDF

Ces bibliothèques fonctionnent entièrement en local et ne transmettent aucune donnée.

Modifications de la Politique de Confidentialité

Nous nous réservons le droit de mettre à jour cette politique de confidentialité. Les modifications seront publiées dans la nouvelle version de l'extension et prendront effet immédiatement.

Contact

Pour toute question concernant cette politique de confidentialité ou l'utilisation de vos données, vous pouvez nous contacter via le Chrome Web Store.

Consentement

En utilisant l'extension Zephyx, vous consentez à cette politique de confidentialité et au traitement de vos données tel que décrit ci-dessus.
